# Weather App

A Python-based weather application that fetches real-time weather data using the OpenWeather API.

## Features
- Real-time weather information
- Temperature, humidity, and weather conditions
- Asynchronous API requests using httpx
- Secure API key management with .env

## Technologies Used
- Python
- asyncio
- httpx
- OpenWeather API
- python-dotenv

## Installation

```bash
git clone https://github.com/Atharva9988/weather-app.git
cd weather-app
pip install -r requirements.txt
